Files with the extension .py are the fundamental units of Python code. These files are plain-text Python source files. An example Python file might look like this:

x = 1
print(x)

A file like the one above can be executed using a Python interpreter, which executes the code. Depending on the development environment, there are different ways to instruct the interpreter to run the file.

.py Files as Modules

Larger Python projects typically consist of multiple .py files that contain reusable code. These files can be imported into other Python scripts to utilize their functionality. In this way, .py files serve as the building blocks for storing, organizing and reusing Python code.
