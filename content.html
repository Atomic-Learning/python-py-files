Files with the extension .py are the fundamental units of Python code. These files are plain-text files which store only Python instructions. An example Python file might look like this:

x = 1
print(x)

Note that the entire contents of the file is made up of Python statements.

A file like the one above can be executed using a Python interpreter, which will execute the lines of code in order. Depending on the development environment, there are different ways to instruct the interpreter to run the file.

.py Files as Modules

Larger Python projects will consist of multiple .py files that contain reusable code. These files can be imported into other Python scripts to utilize their functionality. In this way, .py files serve as the building blocks for storing, organizing and reusing Python code.
